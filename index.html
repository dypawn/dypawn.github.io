Reasarch Introduction

I mainly focus on the following two topics:

  • Graph Machine Learning I focus on designing novel methods to handle the node/edge level tasks in graph machine learning field, including node representation learning, classification, clustering, link prediction, etc. I also focus on capturing and establishing the connections between Graph Signal processing (GSP) and Graph Machine Learning (GML).

  • Multiview Learning
    I focus on designing novel fusion methods to handle several multi-view learning tasks, including multi-view clustering, multi-view feature selection, etc.
